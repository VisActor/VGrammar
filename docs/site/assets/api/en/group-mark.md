# GroupMark

Group Graphics
Inherits `Mark`, all APIs supported by `Mark` can be called on `Component` instances.

## Instance Methods

### appendChild

Add a child graphic

```ts
(mark: IMark | IGroupMark | IGlyphMark) => this;
```

### removeChild

Remove child graphic

```ts
(mark: IMark | IGroupMark | IGlyphMark) => this;
```
